# MyRPGGame-Version-2

## 项目介绍
一个SoulLikeRPG游戏的Demo，基于MyRPGGame-Version-1的代码重构优化，同时加入了更多的功能。
版本1属于快速验证阶段代码，后期功能扩展和优化都是一个很大的问题，版本2会解决这些问题。
Version 1下载地址(已停止开发) ： https://github.com/WilsonUnity/MyRPGGame

## 主要模块

**1. 控制器模块**
* 包含多种控制器（键鼠，手柄）
* （新增）支持控制器之间的动态切换
* 支持玩家自定义映射按键功能（未实现）

**2. 相机模块**
* 自动跟随角色（延迟跟随）
* （新增）防穿墙处理
* （新增）距离自动拉伸
* 锁定敌人目标

**3. （新增）工厂模块**
* 资源工厂
* 角色工厂

**4.  角色动作模块**
* 常规动作（走路，跳跃等）
* 攻击动作（轻攻击，重攻击）
* 防御
* 受伤（**新增**根据攻击者攻击方向改变倾斜角度）

**5.  （新增）AI有限状态机模块**
